# web-java-sample

A small sample project combining a Java source file with a simple web interface.

## Files

- `index.html` — main web page
- `styles.css` — styling for the page
- `Main.java` — Java application source file

## Usage

Open `index.html` in a browser to view the sample web page.

To compile and run the Java program, use:

```bash
javac Main.java
java Main
```
